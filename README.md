# Pneumonia_labeling

This repository contains the 7-Layer Adversarial Consensus Pipeline and Multi-Modal PyTorch datasets for Pneumonia detection using MIMIC-CXR.

## Files
- `data/output_CSVs.zip` contains all the generated Ground Truth Label CSVs (zipped to bypass the 100MB GitHub limit).
- All Python scripts for the generation pipeline and dataset splitting are included.
